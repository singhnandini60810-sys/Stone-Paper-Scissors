A browser-based Rock Paper Scissors game built using HTML, CSS, and JavaScript. This version features a 3-second countdown timer, side-by-side player and computer layout, scoreboard in the center, and start, pause, and restart controls.

Features

3-second countdown timer

Start, Pause, and Restart buttons

Side-by-side player vs computer layout

Score tracking system

Automatic round reset

Custom background and center image

Fully browser-based (no libraries used)

How to Play

Click Start to begin the game.

Select Stone, Paper, or Scissor before the 3-second timer ends.

When the timer reaches zero, the computer makes its choice.

Scores update automatically.

Use Pause to stop the timer or Restart to reset the game.
